# Deckhand Branding

## Logo

The Deckhand logo combines nautical maintenance imagery with a Popeye-inspired sailor palette:

- **Anchor** — the tool of the deckhand, keeping the workspace steady.
- **Spinach leaves** — a nod to Popeye's strength; cleaning should feel effortless.
- **Sailor collar chevrons** — classic nautical red.
- **Colors** — navy, gold, white, and spinach green.

## Assets

| File | Size | Use |
|------|------|-----|
| `assets/logo.png` | 512×512 | Main icon, social preview |

```bash
python3 scripts/generate-logo.py
```
